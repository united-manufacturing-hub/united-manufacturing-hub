# Changelog

This section contains release notes and changelogs for UMH components. Each release includes new features, bug fixes, and any breaking changes you should be aware of.

## Components

### [Management Console](management-console.md)

The web-based interface for managing and monitoring your UMH instances. Includes updates to the dashboard, configuration wizards, data flow visualization, and more.

### [UMH Core](umh-core.md)

The single-container edge gateway that runs on your manufacturing equipment. Includes updates to protocol support, data processing, state machines, and system internals.

## Versioning

UMH follows semantic versioning (SemVer):

- **Major** versions (X.0.0) may include breaking changes
- **Minor** versions (0.X.0) add new features in a backward-compatible manner
- **Patch** versions (0.0.X) include backward-compatible bug fixes
